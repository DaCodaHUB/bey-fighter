# Blade Gamblers
For The Very Serious Juniper Dev Game Jam

How the game works:

Player can be controled by WASD and Spacebar.
WASD controls the player's beyblade's direction and Spacebar would boost the speed of the beyblade.

The timer is set to 45s.
Within 45s, if player can destroy enemy's HP, player wins.
If timer runs out and enemy HP is still not 0, player loses.
